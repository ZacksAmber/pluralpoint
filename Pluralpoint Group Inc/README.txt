Prerequisite:
1. Please make a directory to store 'mdb_ini_Generator.py' and 'mdb_ini_Exporter.py'.
2. Please make a directory named 'mdb' to store mdb files.
3. Search 'mdbtools' online and install it.
---
Instruction:
1. Run 'mdb_ini_Generator.py' on your MacBook or Linux.
2. Run 'mdb_ini_Exporter.py' on your Windows.
P.S: A better solution is sharing a folder through Windows and MackBook/Linux. And let them sync the files and directories inside the folder.
